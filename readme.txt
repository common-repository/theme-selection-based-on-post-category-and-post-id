Theme selection based on post category and post id. If it exists, themefile single-category-(category id).php and single-post-(post id).php is used

== Installation ==

1. Upload 'drhl_templatestuff.php' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it. You can now create the template files to use.
